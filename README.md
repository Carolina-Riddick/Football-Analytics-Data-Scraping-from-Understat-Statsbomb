## Data Scraping from Understat and StatsBomb

This notebook demonstrates how to collect, process, and visualise football data from Understat and StatsBomb.

### Overview

- The project covers key steps involved in extracting and analysing open football data sources.

#### Main Steps

1. Library installation and import
Set up all necessary Python libraries for web scraping, data manipulation, and visualisation.

2. Accessing football data providers
Retrieve match and player data from Understat and StatsBomb.

3. Data manipulation
Process raw data using NumPy and pandas, including the creation of new data frames and calculated variables.

4. Data cleaning and preprocessing
Handle missing values, standardise formats, and prepare data for analysis.

5. Data visualisation
- Generate insights through visual outputs such as:
- Team passing network maps
- Player passing maps
